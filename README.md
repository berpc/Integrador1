## Descripción general

Aplicación web de página única (SPA) desarrollada con **React 18** y **Vite** que permite buscar libros del dominio público, ver su detalle, guardarlos en una lista de deseos personalizada y consultar el historial de libros visitados. Toda la persistencia de datos se realiza en el navegador mediante **localStorage**, sin necesidad de backend propio.
